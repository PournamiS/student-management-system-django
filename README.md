# Student Management System (Django)

This is my first Django project, built to understand role-based authentication and real-world web application flow.

## Features
- Student-only registration
- Login for Admin, Teacher, and Student
- Admin approval for teacher login
- Attendance management by teachers
- Student attendance viewing

## Tech Stack
- Django
- Python
- HTML
- CSS
- Bootstrap
- SQLite / PostgreSQL

## User Roles
- Admin
- Teacher
- Student

## How to Run
1. Clone the repository
2. Create virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Run server: `python manage.py runserver`
